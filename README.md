# Autonomous Driving Car Simulation

This project aims to develop a self-driving car simulation using deep learning and computer vision techniques. The model is trained to navigate a path in a virtual environment using Unity 3D.

## Overview

The primary goal is to leverage TensorFlow and OpenCV for machine learning and computer vision tasks. The simulation is intended to connect with a Unity game for a comprehensive virtual driving experience. Currently, the model is trained successfully but does not connect to the game as the backend connection file is missing.

## Features

- Utilizes TensorFlow for deep learning model development.
- Employs OpenCV for image processing and computer vision tasks.
- Implements various machine learning methodologies to train the model.
- Designed for integration with Unity 3D for a virtual driving experience (integration pending).

## Long-Term Goal

The long-term objective is to fully integrate the trained model with the Unity game, enabling real-time autonomous navigation and seamless interaction within the simulated environment.

## Requirements

- Python 3.x
- TensorFlow
- OpenCV
- Unity 3D (for integration)
